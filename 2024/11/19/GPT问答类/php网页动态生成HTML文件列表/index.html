  1. luci-app-uhttpd ：这是 uhttpd 的配置界面，可以通过 LuCI Web 界面来管理 uhttpd。

  2. luci-ssl-openssl ：让 LuCI 使用 OpenSSL 作为 SSL 后端，确保 HTTPS 访问。

  3. uhttpd-mod-lua 或 uhttpd-mod-ucode （可选）：如果你需要执行其他动态内容，可以选装 Lua 或 ucode 运行接口。
