Mac 快捷键速查 🏠 返回博客首页

截图

⌘ + Shift + 3 截取整个屏幕
⌘ + Shift + 4 选区截图
⌘ + Shift + 4 → 空格 截取指定窗口
⌘ + Shift + 5 打开截图工具
⌘ + Shift + 6 截取触控栏（Touch Bar）

窗口操作

⌘ + Tab 切换应用
⌘ + ~ 切换同一应用窗口
⌘ + M 最小化当前窗口
⌘ + Option + M 最小化所有窗口

Finder 操作

⌘ + N 新建 Finder 窗口
⌘ + Shift + N 新建文件夹
⌘ + Delete 移到废纸篓
⌘ + I 显示信息

文本编辑

⌘ + C 复制
⌘ + V 粘贴
⌘ + X 剪切
⌘ + Z 撤销

系统控制

⌘ + 空格 打开 Spotlight 搜索
⌘ + Option + Esc 强制退出应用
Control + ⬆ Mission Control

浏览器操作

⌘ + T 新建标签页
⌘ + W 关闭当前标签页
⌘ + Shift + T 恢复关闭的标签页

终端操作

⌘ + N 新建终端窗口
⌘ + K 清空终端屏幕
Control + C 终止当前命令
